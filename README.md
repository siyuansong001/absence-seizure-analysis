# EEG analysis

This repository contains example Python scripts for EEG analysis.

## Files

* `01_load_eeg_data.py`
  Loads EEG data from EDF files.

* `02_filter_and_plot_eeg.py`
  Filters EEG signals and plots example traces.

* `03_power_analysis.py`
  Calculates and plots EEG power spectra.

## Requirements

The scripts require Python and the following packages:

```bash
pip install mne numpy matplotlib scipy
```
